
# YAFF

Yet Another Frontend Framework

Dumb React Re-Implementation


## Run Locally

### Clone the project

```bash
  git clone https://github.com/eone666/yaff.git
```

or

```bash
  git clone git@github.com:eone666/yaff.git
```

### Go to the project directory

```bash
  cd yaff
```

### Install dependencies

```bash
  pnpm install
```

### Build packages

```bash
  pnpm build-packages
```

### Start dev server

```bash
  pnpm dev
```
